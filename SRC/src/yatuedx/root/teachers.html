Principal Instructor - Lian Yang

Lian is an achieved top-class software developer. He started his career at Microsoft and spent 15 years there, developing Windows Operating System and other Microsoft commercial products. After leaving Microsoft, he worked as a consultant for some world-class companies such as Costco and Shenzhen Stock Exchange as lead developer or solution architect. He is also passionate about software education and help students as well as professionals become proficient in coding and software designing. He has published several papers at different professional conferences and wrote a book on advanced software development topics. His book "Deep Dive into C++ Semantics and its Performance" won wide praise and recognition in Chinese programming community. He would love to work with you and make you a top-class developer, no matter if you are a beginner or an experienced professional.

  • Programming languages: C++ / Java / JavaScript / C# / SQL
  • Web Architectual Design
  • OOP / Design Patterns
  • Security
  • Performance and Scalability
